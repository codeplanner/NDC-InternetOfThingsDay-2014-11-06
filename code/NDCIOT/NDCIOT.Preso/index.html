Why is "RealTime" important

  • Well...
    • Event driven architecture
    • Should decrease complexity
    • Will increase speed
  • But...
    • A client should never (or rarely) ask for data
    • You should never send data to a client that does not need it
  • So...
    • Simple
    • Fast
    • Saves resources
